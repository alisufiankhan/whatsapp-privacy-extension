## 3. Core Content Sections

### Hero Section
- **Badge:** New Release 1.0
- **Headline:** Privacy for your messages, visible only when you want.
- **Subtitle:** The essential Chrome extension for WhatsApp Web. Blur messages, media, and contacts automatically. Reveal them instantly with a simple hover.
- **Primary CTA:** [Add to Chrome](https://chromewebstore.google.com/detail/whatsapp-privacy-extensio/jbipjgbbfhdgedlpgpngpjoafpncjgdf)
- **Secondary CTA:** Learn More (`#features`)
- **Trust Indicators:** Free • No Data Collection • Open Source
- **Main Visual:** `img/p-e-s.png` (Floating screenshot)

### Features
1. **Blur Messages:** Hide conversation text automatically.
2. **Hide Media:** Blurred images and videos until hover.
3. **Anonymous Contacts:** Blur names and profile pictures.
4. **Instant Toggle:** Global switch and keyboard shortcuts.
